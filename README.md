# Dorothea
A dead simple stow-like program in rust for linux and macos. The binary is `dothea` which is short for Dorothea.

## Disclaimer
I have never used stow, i have only heard people explain it, but it seemed simple so i thought i would make it too. All of bugs in this are Human sourced, no AI was used in the making of this program.

## Usage
`dothea ~/dotfiles` or `dothea`, it will resort to current directory if no directory is passed.
